## 1. `<html>`

**Fungsi:**
Tag utama yang menandai awal dokumen HTML.

**Contoh:**

```html
<html>
</html>
```

---

## 2. `<head>`

**Fungsi:**
Berisi informasi tentang dokumen seperti judul halaman.

**Contoh:**

```html
<head>
<title>Website Saya</title>
</head>
```

---

## 3. `<title>`

**Fungsi:**
Menampilkan judul halaman pada tab browser.

**Contoh:**

```html
<title>Belajar HTML</title>
```

---

## 4. `<body>`

**Fungsi:**
Berisi semua konten yang tampil di halaman website.

**Contoh:**

```html
<body>
Halo Dunia
</body>
```

---

## 5. `<h1>`

**Fungsi:**
Digunakan untuk membuat judul besar.

**Contoh:**

```html
<h1>Judul Utama</h1>
```

---

## 6. `<p>`

**Fungsi:**
Digunakan untuk membuat paragraf.

**Contoh:**

```html
<p>Ini adalah paragraf.</p>
```

---

## 7. `<a>`

**Fungsi:**
Digunakan untuk membuat link.

**Contoh:**

```html
<a href="https://google.com">Buka Google</a>
```

---

## 8. `<img>`

**Fungsi:**
Digunakan untuk menampilkan gambar.

**Contoh:**

```html
<img src="gambar.jpg" alt="gambar contoh">
```

---

## 9. `<ul>`

**Fungsi:**
Digunakan untuk membuat daftar tidak berurutan.

**Contoh:**

```html
<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>
```

---

## 10. `<table>`

**Fungsi:**
Digunakan untuk membuat tabel.

**Contoh:**

```html
<table>
<tr>
<td>Data 1</td>
<td>Data 2</td>
</tr>
</table>
```
